TwitterApi

  1. Home
  2. Modules
  3. TwitterApi
Description

Twitter API methods. This module includes all available API endpoints that can be called from this extension to Twitter server.

This module must run in the browser tab within the Twitter web app context (twitter.com), to populate the correct request headers.
